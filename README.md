## Tip
- 插入图片统一使用 `<div align="center"><img src=""/></div>` 而不是 `![]()`，这样既可以居中，也可以调整图片大小
- 由于 Vuepress2 的严格机制，不要在 markdown 中使用 `<font>`, `<center>` 等废弃标签，尽管在开发环境下不会报错，但是打包时会出错
